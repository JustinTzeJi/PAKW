# Perbelanjaan Asas Kehidupan Wajar (PAKW)
A clearer and more transparent version of [MyPAKW](https://mypakw.dosm.gov.my/) by the Department of Statistics Malaysia, data and algorithm mined from the [official MyPAWK site](https://mypakw.dosm.gov.my/)

1. download and install [uv](https://docs.astral.sh/uv/)
```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```
2. run the marimo notebook
```bash
uv run marimo run notebook.py
```
